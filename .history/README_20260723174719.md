# DRDO Personnel Directory

An NLP-powered intelligent telephonic directory system for DRDO scientists and staff.

## Features
- Semantic search using SBERT (all-MiniLM-L6-v2) + FAISS
- Spell correction and acronym expansion
- Exact keyword matching for lab names and IDs
- Experience-based filtering
- Admin panel with full CRUD operations
- Search analytics dashboard

## Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Backend:** Python FastAPI
- **NLP:** Sentence-BERT + FAISS
- **Database:** MongoDB Atlas

## Setup Instructions

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Environment Variables
Create a `.env` file in the backend folder:
```
MONGO_URI=your_mongodb_connection_string
DB_NAME=drdo_directory
SECRET_KEY=your_secret_key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
ADMIN_USERNAME=admin
ADMIN_PASSWORD=your_password
```
